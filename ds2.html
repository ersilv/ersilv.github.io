Using CNN and Computer Vision to help soccer analytics


This is a work in progress.

The idea is to use Convolutional Neural Network to help visualize the methods described in the post Applying Data Science to Soccer - Part 1 in the actual game video, by using the game and the analysis as inputs to a CNN with the game stream showing the players' markings, pass options, Voronoi cells and more as output.

So far I was only able to highlight the players' locations. Unfortunately the videos perspective is not ideal, a panoramic view of the soccer pitch would be a better option to match with the 2D visualization.
